# Kino biļešu rezervācija

Šī ir interaktīva mājaslapa kino biļešu cenas aprēķinam.

## Funkcionalitāte
- Lietotājs ievada biļešu skaitu
- Aprēķina kopējo cenu
- Ja biļešu skaits >= 5, tiek piemērota 10% atlaide

## Lietošana
1. Ievadi biļešu skaitu
2. Nospied pogu "Aprēķināt"
3. Rezultāts parādās ekrānā
